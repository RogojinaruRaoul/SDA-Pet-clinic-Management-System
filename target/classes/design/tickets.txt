
PCMS-1
As a user of the system, I should be able to add a new veterinarian.
I should be able to specify firstName, lastName, address, speciality.

PCMS-2
As a user, I should be able to view all the vets.
I should be able to view for each vet the id, firstName, lastName, address and speciality.

PCMS-3
As a user, I should be able to delete a veterinarian by id.

PCMS-4
As a user, I should be able to update a veterinarian by firstName, lastName, address and speciality.

PCMS-5
As a user, I should be able to add a new pet.
I should be able to specify race, birthDate, isVaccinated, ownerName.

PCMS-6
As a user, I should be able to view all the pets.
I should be able to view for each vet the id, race, birthDate, isVaccinated, ownerName.

PCMS-7
As a user, I should be able to delete a veterinarian by id.

PCMS-8
As a user, I should be able to update a veterinarian by firstName, lastName, address and speciality.

PCMS-9
As a user of the system, I should be able to add a new consult.
I should be able to specify veterinarian_id, pet_id, date, description.

PCMS-10
As a user, I should be able to view all the consults.
I should be able to view for each vet the id, veterinarian_id, pet_id, date, description.

PCMS-11
As a user, I should be able to update a consult by description.

PCMS-12
Define Hibernate database model.

PCMS-13
Create base repository classes.
